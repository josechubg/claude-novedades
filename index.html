Cómo funciona

Añadir una novedad nueva (cada día)

  1. Abre el archivo data.json
  2. En el array "updates", añade un objeto al principio:

    {
     "id": 99,
     "date": "2026-04-16",
     "title": "Título de la novedad",
     "category": "API",
     "description": "Breve descripción.",
     "link": "https://...",
     "isNew": true
    }
  3. Guarda y sube el archivo a GitHub. Se despliega solo en Vercel/Netlify.

Despliegue en Vercel (gratis, en 2 minutos)

  1. Sube la carpeta claude-novedades/ a un repositorio GitHub
  2. Ve a vercel.com → "Add New Project" → elige ese repo
  3. Framework: Other (es HTML estático puro)
  4. Clic en "Deploy" — obtienes una URL pública al instante
  5. Para actualizar: edita data.json y haz push. Auto-despliega.
